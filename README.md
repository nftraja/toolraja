# toolraja-js
ToolRaja is a curated discovery platform focused on high-quality digital tools, software, and platforms across multiple categories.

The project is designed with a clean structure, fast performance, and scalable architecture, making it suitable for modern web deployment and future expansion.

## Vision
To simplify the exploration of digital tools by presenting them in a structured, intuitive, and category-driven format.

## Tech Stack
- HTML / CSS / JavaScript
- Static architecture (Cloudflare Pages compatible)
- Git-based version control

## Status
This project is currently in active development.
